# Shortping Project

## 개요

- CI, CD툴과 인프라 환경 핸들링을 연습하기 위한 간단한 쇼핑몰 프로젝트
- 화면 개발부터 서버개발, 배포 환경 구축까지 지금 가지고 있는 스킬, 지식 활용을 모두 연습할 수 있는 프로젝트 개발


### 개발 환경

> BackEnd
- Java 17
- SpringBoot 3.2.11-SNAPSHOT

> FrontEnd
- Vite 5.4.8
- React 18.3.1
- TypeScript 5.5.3

> DB
- PostgreSql

> CI/CD
- GitHub Actions
- Docker

#### Libraries
> BackEnd

> FrontEnd

- Zustand
